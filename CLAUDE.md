# air

Self-hosted personalized newsletter — pick sources (Product Hunt, RSS & non-RSS sites, Hacker News…) and get a browser digest. See `README.md`.

**Stack:** Vite · React · TanStack Router/Query · shadcn/ui · Tailwind v4 · Convex (DB/functions/cron). Package manager: **bun** (never npm/pnpm).
