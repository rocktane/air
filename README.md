# air

Self-hosted, personalized newsletter — pick your sources (Product Hunt, blogs with
or without RSS, Hacker News, …) and get a daily/weekly digest in the browser
(email later). An "RSS reader for content that has no RSS feed."

## Stack

- **Frontend**: React + Vite, TanStack Router + TanStack Query, shadcn/ui, Tailwind v4
- **Backend / DB / cron / storage**: Convex (TypeScript)
- **Content extraction**: Convex `"use node"` actions (`rss-parser`, `@mozilla/readability`)

## Getting started

```bash
bun install

# 1) Connect the Convex backend (interactive: logs you in, creates a dev
#    deployment, writes VITE_CONVEX_URL into .env.local, generates convex/_generated)
bunx convex dev         # leave running in one terminal

# 2) Run the web app
bun run dev             # http://localhost:5173
```

> Before Convex is connected the app still boots and shows a **placeholder digest**
> (see `src/lib/placeholder-data.ts`). Live data lands at milestone 3.

## Scripts

- `bun run dev` — Vite dev server
- `bun run build` — typecheck + production build
- `bun run typecheck` — `tsc -b` only
- `bun run lint` — ESLint

## Project layout

```
src/
  main.tsx              # Convex + TanStack Query + Router providers
  routes/
    __root.tsx          # layout + nav
    index.tsx           # / — digest dashboard (placeholder for now)
    sources.tsx         # /sources — manage sources (placeholder)
    settings.tsx        # /settings — schedule (placeholder)
  routeTree.gen.ts      # generated by the TanStack Router Vite plugin
convex/
  schema.ts             # sources / items / settings tables
  sources.ts            # list / add / setEnabled / remove
  digest.ts             # latest — builds the digest from items
```

## Roadmap (POC)

1. ✅ Scaffold — Vite + React + TS + Tailwind v4 + shadcn + TanStack + Convex wired
2. Schema + sources CRUD UI (Convex)
3. Product Hunt fetcher (needs `PRODUCT_HUNT_TOKEN` — `npx convex env set`)
4. RSS fetcher (`rss-parser`)
5. Non-RSS fetcher (hidden-feed discovery → `@mozilla/readability`)
6. Digest dashboard wired to live data + cron refresh

## Notes

- Sites behind Cloudflare / pure-SPA sites may need a Playwright (or Python/trafilatura)
